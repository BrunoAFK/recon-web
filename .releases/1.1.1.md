# v1.1.1

Multi-platform Docker images, secret scanning, dependency patches.

## Highlights

- **Multi-platform Docker images** — CI now builds `linux/amd64` and `linux/arm64` images, supporting both Intel servers and Apple Silicon Macs.
- **Secret scanning** — gitleaks pre-commit hook via husky prevents accidental secret commits.

## Bug Fixes

- **Security deps** — updated vite (CVE-2026-39363, CVE-2026-39364) and basic-ftp (GHSA-chqc-8p9q-pq6q).
- **Husky in Docker** — `prepare` script no longer fails during production builds when husky is not installed.

## CI/CD

- **Multi-platform Docker builds** — release pipeline now uses QEMU + Docker Buildx to produce `linux/amd64` and `linux/arm64` images for all three containers (api, web, cli).
- **Gitleaks pre-commit** — husky runs gitleaks on staged files before every commit to prevent accidental secret leaks.

## Docker images

```bash
docker pull ghcr.io/brunoafk/recon-web/api:v1.1.1
docker pull ghcr.io/brunoafk/recon-web/web:v1.1.1
docker pull ghcr.io/brunoafk/recon-web/cli:v1.1.1
```
